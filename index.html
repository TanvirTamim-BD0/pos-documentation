Cpanel Installation

1. Copy files to a web server

Upload dist.zip to your web server's root (public_html) and extract it there. If you are installing it on subdomain then do extract it in your subdomain's root folder.


2. Setup your Database

Open PHPMyAdmin on your server

Click on the Databases tab. Create a new database and specify a Database name of your choice and click the Create button.

Now on the left, select the database that you have created and click Import in the top menu

Under Import, choose the database SQL file from file/database/pos_p.sql from the folder.


3. Setup environment file

Open .env file on Project.

Write Database Details

DB_DATABASE= your_database_name
DB_USERNAME= your_database_user_name
DB_PASSWORD= your_database_password



Project root folder create .htaccess file and write this code.


<IfModule mod_rewrite.c>
  RewriteEngine on
  RewriteRule ^(.*)$ public/$1 [L]
</IfModule>

<IfModule mime_module>
  AddHandler application/x-httpd-ea-php81 .php .php8 .phtml
</IfModule>

Local Installation

Primary Requirements:

  • XAMPP install verstion 8+ https://www.apachefriends.org/download.html
  • Composer Intsall https://getcomposer.org/download/
1. Setup your Database

Open XAMPP start Apache and Mysql

Open PHPMyAdmin on your localhost

Click on the Databases tab. Create a new database and specify a Database name of your choice and click the Create button.


2. Setup environment file

Unzip Project file.

Opren Terminal and command this line
1. composer update
2. php artisan migrate:fress --seed

Open .env file on Project.

Write Database Details

DB_DATABASE= your_database_name
DB_USERNAME= your_database_user_name
DB_PASSWORD= your_database_password
